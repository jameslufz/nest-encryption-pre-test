# Nest Encryption Pre-Test
Welcome to a repository for `nest encryption pre-test`

## First, install dependencies
After you clone this repo to your local, let's install dependencies first.
```bash
npm install
```

## Start the application
Let's build the project and start it.
```bash
npm run build && npm run start
```
When app started, your localhost will be listening on port `3000`

## Document (Swagger)
For document you can access to [Document](http://localhost:3000/document) or [http://localhost:3000/document](http://localhost:3000/document).

## For unit test
You can run a command following below on your terminal:
```bash
npm run test
```
